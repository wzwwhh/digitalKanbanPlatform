## 📋 测试前准备

### 1. MySQL 数据库

执行 `doc/test_mysql_init.sql`，创建 `kanban_test` 数据库，包含 3 张表：

| 表名 | 内容 | 行数 | 用途 |
|------|------|------|------|
| `sales` | 月度×品类销售统计 | 20 | 折线图、柱状图、饼图测试 |
| `orders` | 订单明细 | 25 | 表格、排行榜、条件查询测试 |
| `region_stats` | 各省销售统计 | 31 | 地图组件测试 |

> **所有字段都有 COMMENT 注释**，用于测试"字段注释自动识别"功能。

```bash
mysql -u root -p < doc/test_mysql_init.sql
```

### 2. 后端配置

当前 `db_connector.py` 默认连的是 SQLite。要测 MySQL，需要在**数据源管理里手动添加 MySQL 类型的数据源**（前端会传连接参数）。

确保已安装 pymysql：
```bash
pip install pymysql
```

### 3. 启动服务

```bash
# 终端 1: 后端
cd backend && python -m uvicorn app.main:app --reload --port 8000

# 终端 2: 前端
cd frontend && npm run dev
```

### 4. 初始化 SQLite 示例库（如果没有的话）

```bash
cd backend && python init_sample_db.py
```

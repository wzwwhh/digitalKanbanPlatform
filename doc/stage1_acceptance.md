# AI 看板平台 - 第一阶段验收说明

## 验收目标
验证项目已经具备一个可稳定试用的 MVP 原型能力，覆盖：首页创建项目、进入工作区、编辑组件、主题切换、保存/恢复、AI 操作、导出原型。

## 验收范围
- 项目首页可新建项目、打开已有项目
- 支持两种模式：从数据开始 / 从模板开始
- 进入工作区后可添加组件
- 组件支持拖拽、缩放、选中、属性编辑
- 主题可切换，并在打开项目后恢复
- 支持保存/恢复项目状态
- AI 面板可执行基础指令
- 支持导出 HTML 原型并正常打开
- 数据源配置面板可完成 API 探测流程

## 验收步骤
1. 打开首页
2. 创建一个新项目
3. 选择数据先行或设计先行模式
4. 若为数据先行，先配置一个 API 数据源
5. 返回工作区并添加一个组件
6. 拖拽组件位置，调整组件大小
7. 在属性面板修改组件属性
8. 切换主题，确认样式立即变化
9. 点击保存，刷新页面后重新打开项目
10. 切换到 AI 面板，输入基础指令并确认命令执行成功
11. 点击导出，下载 HTML 文件并在浏览器中打开

## 通过标准
- 页面不崩溃，主流程可完整走通
- 保存/恢复有效
- 导出文件可正常打开并渲染
- AI 命令可执行
- 主题切换和组件编辑正常
- 数据源 API 探测流程可用

- DB 后端直连仍在继续完善
- 数据绑定能力已具备基础入口，后续还会继续增强
